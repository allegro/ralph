Development Get involved in improving Ralph

The main Ralph repository is published on Github. The project is licensed under the Apache License, Version 2.0. It's very easy to step up and help with developing future versions of Ralph. Everybody can contribute by:

  • writing code, especially plugins for hardware we don't use ourselves
  • writing documentation
  • creating translations
  • testing
  • reporting bugs

In each case, contributors remain copyright holders. We only ask them to formally confirm the terms under which they made their contribution available by signing a Contributor Agreement (PDF, Microsoft Word).

Fork us on Github

Getting Help Contact the developers

There are several ways to contact the people involved in Ralph development:

  • Wiki
  • Bug tracker
  • #ralph IRC channel on FreeNode
  • Ralph mailing list on Librelist. To subscribe simply send an empty message to ralph@librelist.com.
